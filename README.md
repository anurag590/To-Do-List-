# 📝 To-Do List Web App

A **simple**, **responsive**, and **interactive** To-Do List web application built using **HTML5, CSS3, and JavaScript**. It allows users to manage daily tasks easily with a clean and modern user interface.

## 🔧 Features

- ✅ Add tasks dynamically  
- 🗑️ Delete tasks individually  
- ⚠️ Input validation for empty tasks  
- 🎨 Responsive UI with background image  
- 💬 Motivational quote shown on every task deletion

## 🚀 How It Works

1. Type your task into the input field  
2. Click “Add” to insert it into your task list  
3. Click “Delete” to remove it and get a motivational boost 😉
